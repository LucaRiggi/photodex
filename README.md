# PhotoDex

PhotoDex is an Android application built using Kotlin and Jetpack Compose. It serves as a stock photo gallery that allows users to browse, search, favourite, and download high-quality royalty-free images. 


## Authors

- Luca Riggi - Designer/Developer
- Gursimran Kaur - Documentation
- Stanislav Nehretskyi - Designer/Developer




## Run Locally

Clone the project

```bash
  git clone https://github.com/LucaRiggi/photodex.git
```

Open the project in Android Studio

Sync project

Run the app
